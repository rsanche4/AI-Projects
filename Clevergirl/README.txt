***************************************
*  A simple implementation of a       *
*  bot that learns from users         *
*  different replies.                 *
*                                     *
*  Clevergirl takes a response,       *
*  checks in a json file if it        *
*  knows the answer to that           *
*  response, otherwise it learns      *
*  from the user what it should say.  *
*                                     *
*  A very simple bot that gradually   *
*  grows as more people teach it      *
*  new responses.                     *
***************************************
